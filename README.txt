# Super Trunfo - Comparação de Cartas

Este programa em C permite comparar duas cartas de países (ou cidades) com base em diferentes atributos usando um menu interativo.

## **Atributos disponíveis para comparação**

1. População
2. Área (km²)
3. PIB (bilhões de reais)
4. Pontos Turísticos
5. Densidade Demográfica (hab/km²) — neste caso, menor valor vence

## **Como compilar**

No terminal, navegue até a pasta do projeto e digite:

```bash
gcc super_trunfo.c -o super_trunfo
